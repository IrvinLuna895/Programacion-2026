En esta versión se implementa encapsulamiento haciendo privados todos los atributos y accediendo a ellos mediante getters. La clase Cuenta mantiene la lógica y validaciones devolviendo valores booleanos, mientras que Menu gestiona la interacción con el usuario. Además, se muestra el saldo actualizado después de cada operación. Estos cambios mejoran la seguridad y el diseño del sistema.
